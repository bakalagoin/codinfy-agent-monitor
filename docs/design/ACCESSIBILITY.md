# Accessibility

- Text and state colors target WCAG AA contrast on every dark surface.
- Color is never the only state cue; badges include text.
- All controls use native buttons, inputs, labels, and focus indicators.
- Confirmation dialogs use `role="dialog"`, `aria-modal`, a labelled title, and predictable focus order.
- Tables preserve a textual mobile layout; important identity fields are never hidden simultaneously.
- Reduced-motion users receive no pulse or transition dependency.
