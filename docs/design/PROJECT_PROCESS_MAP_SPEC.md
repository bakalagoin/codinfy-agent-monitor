# Project Process Map

Route `/process-map` uses `monitor.project_process_map`. Processes are grouped only when a project path is actually detected. Each group shows PIDs, unique ports, frameworks, resource totals, risk, and protection. Unknown projects remain in an explicit unassigned group.

Parent-child links are displayed only when both live PIDs are present in the inventory.
