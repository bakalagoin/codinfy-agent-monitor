# Design system

## Foundations

- Background: `#070A0F`; raised surface: `#0A111C`; strong surface: `#0D1724`.
- Hairline: `rgba(126, 176, 214, .18)`; cyan `#42D5FF`; green `#38E6A1`; amber `#FFC96B`; red `#FF647C`; violet `#A678FF`.
- UI text uses Inter/Segoe UI; operational values use Cascadia Code/Consolas.
- Corners stay restrained: 8–14 px for controls and panels. Blur is reserved for the shell and confirmation overlays.
- Motion communicates state only: live pulse, progress change, refresh, modal entry. Respect `prefers-reduced-motion`.

## Product rules

No decorative number, fake chart, invented agent, fabricated usage value, or simulated success state is allowed. Values carry their provenance: live, official, estimated, unavailable, or last updated.

The official Codinfy SVG/PNG assets in `assets/brand/` are used unchanged. Identity and attribution never collapse behind an unlabeled icon.
