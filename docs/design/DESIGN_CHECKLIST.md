# Design release checklist

- [ ] Official Codinfy mark is visible and unmodified.
- [ ] `/codinfy`, MCP name, creator, founder, site, footer, and social credits are present.
- [ ] Every metric has source and state; no fake data exists.
- [ ] Loading, empty, error, unavailable, protected, and confirmation states render.
- [ ] Stop, Force Kill, install, rollback, and restore are server-gated.
- [ ] Keyboard, focus, contrast, reduced motion, mobile, and zoom are checked.
- [ ] Secrets remain redacted in HTML, JSON, logs, and screenshots.
- [ ] Core tests, full build, server route tests, and browser QA pass.
