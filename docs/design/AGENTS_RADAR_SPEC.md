# Agents Radar specification

The radar summarizes registered agents only. Position is a stable visual distribution, not geospatial or performance data. Color encodes state: green active, blue running/reading/writing, grey idle/done, amber blocked, red error. The adjacent list remains the accessible source of truth.

Empty state: “No agent registered.” Never generate sample agents in production.
