# Information architecture

1. Mission control: dashboard, agents, workflow, tasks, context, limits, timeline.
2. AI efficiency: Smart Model Router and AI Credit Saver.
3. Developer tools: Node Server Monitor, Port Conflict Resolver, Project Process Map, Resource Guard, MCP Health Doctor.
4. Release safety: Git, files, tests, build, review, security, reports.
5. Continuity: Session Recovery, Notifications, Backup & Restore.
6. Settings: MCP Update Center, Release Notes, language, experience level, attribution, About.

Every route resolves inside the same local-only application shell. `/codinfy` and `/codinfy-agent-monitor` remain aliases of Mission control.
