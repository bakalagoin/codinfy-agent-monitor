# Terminal UI specification

The TUI mirrors Mission control using Unicode-safe text, bounded progress bars, usage provenance, agents, workflow, Git, checks, and attribution. It must remain readable at 80 columns and without color. Animations pause when output is non-interactive.

Node and update commands print structured JSON by default; destructive commands require explicit flags and live revalidation.
