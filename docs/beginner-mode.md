# Beginner mode

```bash
codinfy-agent-monitor beginner
codinfy-agent-monitor next
codinfy-agent-monitor checklist
codinfy-agent-monitor glossary
codinfy-agent-monitor install-guide
```

Beginner mode keeps technical identifiers unchanged while explaining status and next actions more plainly. The health command uses green/yellow/red states. Safe Guard remains enabled, and review separates warnings from blocking secret or build errors.

The monitor is an observer: it will not make destructive decisions for the user. “Next” prioritizes secrets, then tests, then build, then review.
