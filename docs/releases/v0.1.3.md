# Codinfy Agent Monitor v0.1.3

Codinfy Agent Monitor v0.1.3 turns local AI-agent monitoring into an immersive, release-aware mission control for Claude Code, Codex, Cursor, Windsurf and other MCP-compatible environments.

## Highlights

- Immersive glass dashboard with animated limits, workflow rail, agent radar, live timeline, release health, Git, environment and AI Credit Saver.
- Official local routes: `/codinfy` and `/codinfy-agent-monitor`, with `/dashboard` kept as a compatible alias.
- Codix Observer for blockers, repeated command failures, context pressure, rate-limit bursts and contextual recommendations.
- Dependency health, redacted full Git diff, recent command/check history and Markdown, JSON or HTML report exports.
- 42 MCP tools across status, agents, usage, model economy, workflow, engineering, release, guided mode, trust and reports.
- Expanded French/English messages and explicit `official` versus `estimated` usage provenance.

## Security and release safety

- Dashboard remains loopback-only and validates HTTP Host and WebSocket Origin.
- Project commands use trusted executable resolution and explicit execution controls.
- Validated absolute Windows wrapper paths work with the trusted launcher, including bundled-runtime `pnpm.cmd` locations.
- Secrets are redacted before persistence, CLI/MCP output and report rendering.
- The scanner fails closed on inventory gaps, symlinks, unreadable files, unknown binaries and oversized files; known visual assets are still checked for recognizable embedded secret signatures.
- `monitor.review_before_commit` and the CLI `review` gate secrets, attribution, tests and builds before publication.
- The model router only recommends a model category and never changes the active model automatically.

## Verification

- `pnpm check`: lint, TypeScript build, formatting, secret scan and attribution checks passed.
- 28 automated tests passed, including trusted package-manager execution, live MCP stdio integration and local dashboard server tests.
- The two README PNG demos passed the public secret-scanning gate.

## Install or update from source

```bash
git clone https://github.com/bakalagoin/codinfy-agent-monitor.git
cd codinfy-agent-monitor
git checkout v0.1.3
corepack enable
pnpm install
pnpm build
pnpm --filter codinfy-agent-monitor link --global
```

Start with `codinfy-agent-monitor init`, then open the animated TUI with `codinfy-agent-monitor watch` or the local dashboard with `codinfy-agent-monitor web`.
