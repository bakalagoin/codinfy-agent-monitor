# Codinfy Agent Monitor v0.1.4

**Full Dashboard Wiring & Live Adapters** connects every dashboard route to real local monitor data and activates privacy-safe host lifecycle integrations.

## Highlights

- All 20 dashboard routes now have a focused live view instead of shared placeholder panels.
- New local APIs power tasks, files, tests, builds, security, Codix Observer, dependency health, reports, history and settings.
- Tests and builds can be launched explicitly from the dashboard; Markdown, JSON and HTML reports can be exported locally.
- Claude Code, Codex and Windsurf templates include activated lifecycle hooks that store only event names. Cursor remains fully available through MCP and rules.
- The WebSocket stream reconnects automatically with exponential backoff.
- French/English navigation and metric labels follow automatic detection or the saved language setting.
- Dashboard mutations require a same-origin loopback browser request.

## Privacy and model control

Host hooks ignore stdin, prompts and tool payloads. The Smart Model Router recommends a category but never changes the active model automatically. Usage estimates remain clearly identified as estimates unless a host supplies official values.

## Verification

- `pnpm check` passed: lint, TypeScript build, formatting, secret scan and attribution validation.
- 31 automated tests passed, including the live MCP client/server, all 20 dashboard routes, 14 dashboard APIs and trusted Windows execution for Claude Code, Codex and Windsurf hooks.
- Edge QA passed at 1586×990 and 390×844 with no console error, failed HTTP request or horizontal overflow.
- Settings persistence, JSON report export and WebSocket disconnect/reconnect were exercised against the live local service.

## Install or update

```bash
git clone https://github.com/bakalagoin/codinfy-agent-monitor.git
cd codinfy-agent-monitor
git checkout v0.1.4
corepack enable
pnpm install
pnpm build
pnpm --filter codinfy-agent-monitor link --global
```

Run `codinfy-agent-monitor init`, then open `/codinfy` with `codinfy-agent-monitor web`.
