# Security policy

Please report vulnerabilities privately through GitHub security reporting. Do not open a public issue containing a token, key, password, private log, or exploit against a live system.

Supported line: V1 (`0.1.x`). Run `codinfy-agent-monitor secrets` and `review` before publication. See `docs/security.md` for the local threat model.
