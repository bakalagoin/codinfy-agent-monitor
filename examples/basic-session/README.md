# Basic session

```bash
codinfy-agent-monitor init
codinfy-agent-monitor metric context 40 --source estimated
codinfy-agent-monitor event session.note "Starting the documented example"
codinfy-agent-monitor status
codinfy-agent-monitor export
```

MCP clients can call `monitor.register_agent`, `monitor.create_task`, and `monitor.update_task` to make the live TUI reflect their workflow.
